# Teton_LiDAR_paper

- All code and scripts used for LiDAR data analysis and figure generation are stored in the `Lidar_Data_analysis` directory.  
- Code used to calculate the percentage of snow remaining in each watershed using Sentinel-2 satellite imagery is stored in the `Year_of_year_Snow_percentage_using_Sentinel2` directory.  
- `ERA5_teton_analysis.ipynb` contains the code for the “Meteorological Drivers of Glacier Change in the Teton Range” analysis (Fig. S5).
